# Python-Project
My Project on Python

Это мой учебный проект по питону, представляющий собой сайт со статьями из Хабра. 
На сайте осуществляется реализация алгоритмов регистрации/авторизации. 
После регистрации и последующей авторизации пользователь может просматривать статьи, комментировать их и также удалять комментарии. 
Статьи генерируются посредством парсинга Хабра.
Целью проекта является закрепление изученного материала
Инструментарий: Python, Django, реляционные БД, HTML, CSS
